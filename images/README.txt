PAXO SITE IMAGES
================
Drop your own assets here with these exact filenames.
The site references them; missing files fall back to CSS gradients (no broken images).

  hero-bg.jpg      — Pittsburgh skyline / bridge. Wide, ~2400x1200. Used behind the hero.
  driver.jpg       — Driver in car, smiling. ~1200x1200 square-ish. Left card / driver page.
  rider.jpg        — Rider with phone. ~1200x1200 square-ish. Right card / rider page.
  app-preview.png  — Phone mockup showing the app. Transparent PNG preferred. ~800x1600.
  logo.svg         — PAXO wordmark + pin mark. Replaces the built-in CSS logo if present.

Tips:
- Keep files under ~400KB each so the page stays fast. Use JPG for photos, PNG/SVG for logo.
- After adding: git add images/ && git commit -m "Add site imagery" && git push
